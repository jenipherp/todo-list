# Role Based Task Management Web App

A Spring Boot web application developed using Spring web and Spring JPA, H2 Database.
A role-based task management web application built with Java, Spring Boot, and Spring Security. Users can register, log in, and manage personal to-do items with full CRUD functionality. Built with Spring MVC, Spring Data JPA, and an H2 in-memory database. Tech Stack: Java · Spring Boot · Spring Security · Spring JPA · H2 · Maven · Bootstrap How to run: Clone the repo → mvn spring-boot:run → visit http://localhost:8080
